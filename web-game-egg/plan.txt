网页版鹅类下蛋射击游戏（第一版）实施计划

画布与布局
- 画布固定 390×844（iPhone 常见比例），居中显示并自适应缩放。
- 三区域：顶部鹅区 / 中部战斗区 / 底部信息区。

回合与波次
- 单关卡 3 波，每波随机 5–15 只怪物。
- 每波清空后进入下一波；第 3 波清空即胜利。

发射节奏
- 每次发射生成 5 枚蛋（5 只鹅同步）。
- 下一次发射必须等待当前批次所有蛋都“落地/命中/消失”。

瞄准模式
- 默认手动模式：鼠标拖动确定目标点，松开立即发射。
- 自动模式：仅在切换开启时系统接管锁定最近怪物并自动发射。

怪物逻辑
- 从底部随机 X 生成，匀速向上。
- 任意怪物到达鹅区 → 失败。

弹道与目标切换
- 蛋沿直线飞行；目标消失后，同批未命中蛋锁定最近存活怪物并更新角度。

UI
- 开始、重新开始、退出。
- 模式切换（手动/自动）。
- 底部显示：当前波次 / 剩余怪物。

参数配置（可在配置对象内调整）
- 单次下蛋数量：5
- 单颗蛋伤害：10
- 怪物移动速度：2 像素/帧
- 下蛋频率：2000ms（自动模式使用）
